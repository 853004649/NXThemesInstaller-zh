# NXThemesInstaller-zh ✨

Switch 主题安装器 **简体中文汉化版**

---

## 项目介绍

本项目是 [NXThemesInstaller](https://github.com/exelix11/SwitchThemeInjector) 的中文汉化版本，基于原作者 `exelix11` 的开源代码进行本地化修改，方便中文用户使用。

### 原作者项目

- 原仓库：[exelix11/SwitchThemeInjector](https://github.com/exelix11/SwitchThemeInjector)
- 作者：`exelix11`

## 一键下载（推荐）

打开终端 / Git Bash 运行以下命令，即可直接下载汉化版 NRO：

```bash
curl -sSL https://raw.githubusercontent.com/853004649/NXThemesInstaller-zh/main/download.sh | sh
```
